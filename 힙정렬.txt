힙 구조 
완전이진 트리에서 최솟값이나 최대값을 빠르게 찾아내기 위해 사용
완전이진트리란 왼쪽에서 부터 순서대로 값이 들어가는 형태
                                  1
		2	   3
  	        4       5	6       7

힙 생성 알고리즘은 특정학 노드의 두 자식중에서 더 큰 자식과 자신의 위치를 바꿈 


#include<stdio.h>

int number= 9;

int heap[9]={7,6,5,8,3,5,9,1,6};


int main(){


for(int i=1;i<number;i++)
{
	int c=i;
	do{
	  int root =(c-1)/2;
	  if(heap[root] < heap[c])
		{
			int temp =heap[root];
			heap[root]=heap[c];
			heap[c]=temp;
		}
		c=root;

}while(c!=0)

}
}





}
